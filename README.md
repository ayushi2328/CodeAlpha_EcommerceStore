# Ecommerce Store

A Full Stack Ecommerce Store built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features

- User Registration & Login
- Authentication
- Add Products
- Edit Products
- Delete Products
- Shopping Cart
- Search Products
- Logout System
- Responsive UI

## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Node.js
- Express.js

Database:
- MongoDB
